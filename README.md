## Overview
This repository contains a collection of advanced Generative AI solutions, from prompt design to production deployment (MLOps).

## Project Roadmap
The projects in this repository are structured sequentially to build complexity step-by-step. It is recommended to explore them in the following order:

1.  **[reader_digest](./reader_digest)**
    * *Focus:* Displays a summary of web pages
2.  **[digital_brochure](./digital_brochure)**
    * *Focus:* automates the creation of marketing collateral
3.  **[03_Advanced_RAG](./03_Advanced_RAG)**
    * *Focus:* Hybrid search, re-ranking, and context window optimization.
4.  **[04_Fine_Tuning](./04_Fine_Tuning)**
    * *Focus:* PEFT (Parameter-Efficient Fine-Tuning) and LoRA implementations on open-source models.
5.  **[05_LLM_Agents](./05_LLM_Agents)**
    * *Focus:* Building autonomous agents using frameworks like LangChain or LangGraph.
6.  **[06_MLOps_Deployment](./06_MLOps_Deployment)**
    * *Focus:* Model serving, containerization (Docker), and monitoring via tools like MLflow or LangSmith.
